## ğŸ¯ Key User Journeys

### Journey 1: "I want to track my coding activity"

**Steps:**
1. Land on homepage
2. Login with GitHub
3. Sync activity
4. View dashboard with stats
5. Check streak
6. Set goal to maintain streak

**Success Metric**: User syncs activity and returns daily

---

### Journey 2: "I want to compete with others"

**Steps:**
1. Login
2. View leaderboard
3. See own rank
4. Check top users
5. Get motivated
6. Code more to improve rank

**Success Metric**: User checks leaderboard regularly

---

### Journey 3: "I want to follow friends"

**Steps:**
1. Login
2. Search for friends
3. View their profiles
4. Follow them
5. See their activity in feed
6. Get inspired by their commits

**Success Metric**: User follows 5+ people and checks feed daily

---

### Journey 4: "I want to showcase my profile"

**Steps:**
1. Login
2. Sync activity
3. Build up streak
4. Share profile link
5. Others view public profile
6. Gain followers

**Success Metric**: Profile views and follower growth
